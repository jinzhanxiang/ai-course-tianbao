🚀 安装步骤（完全离线）

1 获取商汤 API Key（需联网完成，一次性）

访问 sensenova.cn → 注册 → 控制台 → API Key 管理 → 创建 Key

免费额度：DeepSeek V4 Flash 模型，每分钟 60 次调用，每天 10000 次

2 解压安装包

将 hermes-installer/ 文件夹复制到你的电脑上（如 D:\hermes-agent\）

安装包已内含：Python 3.11.9 嵌入式版 + 29 个 Python 依赖包 + pip

3 双击运行 setup.bat

脚本自动完成：解压内置 Python → 安装 pip → 创建虚拟环境 → 安装所有依赖 → 创建启动脚本

全程无需联网！所有依赖包已预下载在 wheels/ 目录中。

安装完成后会自动打开 config.yaml 供你填入 API Key。

4 编辑 config.yaml，填入商汤 API Key 
model:
  api_key: "sk-你的商汤API密钥"  # 改这里
5 双击 "启动Hermes.bat" 开始使用

进入交互式对话界面，输入问题即可获得回复。

🪶 连接飞书（可选）

  1. 打开飞书 → 目标群聊 → 设置 → 群机器人
  2. 添加自定义机器人 → 复制 Webhook URL
  3. 编辑 config.yaml，将 Webhook URL 填入 feishu.webhook_url
  4. 在 Hermes 对话中生成内容后，输入 /feishu 即可推送到群里
⚠️ 注意：飞书自定义机器人仅支持向群内推送消息。如需接收消息并自动回复，需要创建飞书应用并配置事件订阅。详见飞书开放平台文档。

🔧 常见问题

Q: 安装时需要联网吗？

不需要！安装包已内置 Python 3.11.9 和全部 29 个依赖包。安装过程完全离线。
使用时需要网络访问 token.sensenova.cn（商汤 API 地址，国内可直接访问）。

Q: Python 未找到 / 需要安装 Python 吗？

不需要！安装包内置了完整的 Python 3.11.9 嵌入式版（python-embed.zip），setup.bat 会自动解压和配置。
整个安装过程不需要你系统上有 Python，也不需要联网下载任何东西。

Q: 解压失败怎么办？

Windows 7 或旧版 Windows 10 可能没有 PowerShell 5.0+。请手动：

1. 右键 python-embed.zip → 解压到当前文件夹 → 重命名文件夹为 python
2. 编辑 python\python311._pth，取消 #import site 的注释（删掉 #）
3. 在文件末尾加一行: Lib\site-packages
4. 重新双击 setup.bat

Q: 商汤 API 调用失败？

检查 config.yaml 中的 api_key 是否正确。登录 sensenova.cn 控制台确认 Key 状态。
API 地址: https://token.sensenova.cn/v1（国内访问无障碍）

Q: 安装包太大，有没有精简版？

当前 23MB 已是最小版本。如果你已安装 Python 3.11+，可以只用 hermes_lite.py + requirements.txt，手动 pip install（需要使用国内镜像加速）：

pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

Q: 如何升级到新版本？

替换 hermes_lite.py 和 config.yaml.template 即可。如 requirements.txt 有变化，重新运行 setup.bat。
