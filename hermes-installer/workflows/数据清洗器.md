# 数据清洗器

## 功能
识别数据质量问题，自动清洗去重，生成清洗报告。

## 使用方法
1. 在 Hermes 对话中输入 `/清洗`
2. 描述数据问题或粘贴数据样本
3. Agent 分析问题类型并给出清洗方案
4. 执行清洗并输出对比结果

## Prompt 模板
```
你是一个数据清洗专家。请分析以下数据的问题：

检查项：
- 重复行检测
- 缺失值统计
- 日期格式标准化
- 异常值检测（超过 3σ）
- 字符串规范化（空格、大小写、全角半角）

输出：
1. 问题统计表
2. 清洗规则
3. 清洗前后对比
4. 数据质量评分
```

## 适用场景
- Excel 报表预处理
- 客户名单去重合并
- 财务数据格式标准化
- 调研数据清理
