# 日报生成器

## 功能
自动汇总工作内容，生成结构化日报，支持推送到飞书群。

## 使用方法
1. 在 Hermes 对话中输入 `/日报`
2. 输入今日工作内容（或粘贴聊天记录/邮件摘要）
3. Agent 自动生成格式化日报
4. 输入 `/feishu` 推送到飞书群

## Prompt 模板
```
你是一个专业的日报生成助手。请根据以下工作内容生成日报：

格式要求：
- 已完成事项（带进度%）
- 进行中事项（含预计完成时间）
- 明日计划（按优先级排列）
- 需要协调的事项

请用专业、简洁的语言，突出关键进展和风险点。
```
