# 信息爬取器

## 功能
输入目标 URL 或网页内容，自动提取关键信息并结构化输出。

## 使用方法
1. 在 Hermes 对话中输入 `/爬取`
2. 粘贴网页 URL 或网页内容
3. Agent 自动提取标题、关键数据、核心观点
4. 输出结构化 JSON/Markdown

## Prompt 模板
```
你是一个网页信息提取助手。请从以下内容中提取：

1. 来源信息（标题、URL、发布日期）
2. 关键数据（所有数字、百分比、日期、金额）
3. 核心观点（3-5 条，每条不超过 50 字）
4. 可信度评估（高/中/低 + 理由）

输出格式：结构化 Markdown，关键数据用表格呈现。
```

## 适用场景
- 竞品官网动态监测
- 行业政策文件摘要
- 公告/新闻关键信息提取
- 招投标信息结构化
