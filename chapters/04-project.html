📄 投资请示起草

project 负责起草投资请示文档框架，调用 research 出行业分析、data 出财务数据、report 做最终排版。main 负责调度各环节协同。

📅 节点跟踪

每个项目关键节点（立项/尽调/投决/投后）都有时间戳记录。决策者随时问 project，项目当前在哪一步，一目了然。

🔗 GP/LP 双线

盈科汇金等多个基金项目涉及 GP（普通合伙人）+ LP（有限合伙人）双线管理。project 负责维护 GP→LP 关系和出资结构。

🤝 跨 Agent 协调

project 接到 main 分派 → research 做研究框架 → data 清洗材料 → report 排版输出。全程不需要决策者介入。

🤝 project 与 main 的协同关系

场景 main 的动作 project 的动作
新项目立项 main 拆解任务，分派给 project project 建档、起草立项建议
项目需要研究 main 调度 research 协同 project 提供研究框架需求
项目需要材料 main 调度 data 协同 project 整理材料清单
需要输出文档 main 调度 report 协同 project 提供文档框架和内容要点
投决前夕 main 监控全流程进度 project 汇总所有材料，通知决策者
