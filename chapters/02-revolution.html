🤖 什么是智能体？

先看清楚这场变革 —— 智能体时代 vs 传统时代，到底变了什么？

变革大背景：我们正在经历 “软件 2.0” 的变革。从“手工生产” → “智能工厂” 需要三步：云计算 (2010s) → 大模型 (2023) → 智能体 (2024+)。

🌍 拿 3 件事看变革

📧 发一封邮件

传统：打开 Outlook → 点\"新建\" → 输入收件人 → 写主题 → 写正文 → 点\"发送\"

📑 出一份尽调报告

传统：50 页 Word → 手工查财报 → 手工录数据 → 手工改格式
智能体：一句话：\"出启源芯尽调报告\" → 7 维度并行 90 分钟出 80 页成稿

🧾 清洗 13 份研报

传统：实习生请假 3 天手工 OCR + Excel
智能体：data agent 2 小时 → 79 实体 / 110 指标 / 178 逻辑/ 自动入库

本质区别：传统 LLM 是“问答机器”，智能体是“数字员工”。这不是进化，是范式跃迁——就像从马车到汽车，不是马车快一点。

① Chatbox 开始 — 人与 AI 的第一次对话

User:
天保控股是哪一年成立的？
AI:
天保控股成立于2009年...
起点：2017 年 Google 发表《Attention Is All You Need》，提出 Transformer 架构。2022 年 11 月 OpenAI 发布 ChatGPT，5 天 100 万用户、2 个月 1 亿用户，史上增长最快的消费级应用。

本质：一个人类输入文本框、一个 LLM、一个文本输出框。一句话能聊，一件事做不了。

② Chatbox 的主要特点 — 三句话概括

🎯 单轮对话 / 多轮上下文
你问一句它答一句。能记住本轮聊过什么，但关掉就忘。没有跨会话记忆，没有跨任务能力。
📚 只能「答」不能「做」
能写文案、写代码、翻译、总结。不能读你的文件、不能跑你的代码、不能改你的数据库。你看到的是文字，不是结果。
🧠 知识停在训练日
训练完什么就会什么，不知道训练后发生的事（幻觉就是这个原因）。要新知识？只能重新训练或外挂 RAG。

③ 智能体 Agent — 给 LLM 装上手脚

核心变化（2024 年起）：LLM 不再只是“答题”，而是“接目标→拆任务→调工具→交付结果”。

公式：Agent = LLM（大胼） + Tools（双手） + Memory（记忆） + Planning（规划）

三个代表性事件：
  • Devin（2024.3）— 首个被广泛关注的 AI 软件工程师，能独立接单完成 bug fix
  • AutoGPT / BabyAGI（2023 Q4）— 证明了 LLM 可以自主拆解目标、循环执行
  • Anthropic Computer Use / OpenAI Operator（2024 Q4）— Agent 能像人一样操控浏览器和桌面

④ 智能体的下一步 — 四个方向已经在走

🌐 1. 多 Agent 协同
不再是“一个 Agent 干一切”，而是 main 调度 + 4 专业 Agent（research/data/project/report）。天保已在路上。
🧠 2. 长期记忆 / 个性化
RAG + 向量库 + 持续学习。OpenClaw Hermes 就是这个方向：记住公司、项目、同事的偏好。
🤖 3. 具身智能 / 物理 Agent
从虚拟走到物理：自动驾驶、人形机器人（特斯拉 Optimus、Figure 02）。2025 年是起点。
🏢 4. 垂直领域 Agent 公司
每个行业都会跑出“投研 Agent 公司 / 尽调 Agent 公司 / 法务 Agent 公司”。天保走的是“投资垂类 Agent 团队”。
📍 我们的位置：Chatbox 阶段 → 已过 ✅｜智能体阶段 → 在用 ✅｜下一步（多 Agent 协同 + 长期记忆）→ 正在进行中 ⏳｜具身智能 → 观望中 👀

🏗️ 智能体的最小架构

🧠 大脑（LLM）

负责推理、决策、规划。MiniMax / Kimi / Claude / Deepseek / 小米 Mimo，多模型融合。

🔧 双手（Tools）

读 PDF、写文件、跑 Python、调数据库。Function Call 协议。

📝 记忆（Memory）

短期上下文 + 长期向量库。OpenClaw 用 Qdrant + bge-m3。

📋 计划（Planning）

CoT 思维链 + ToT 思维树 + ReAct 推理-行动循环。

🧠 main · 总调度

职责定位：最高指挥 · 接收主公指令 · 拆分 · 分派 · 验收

  3. 📡 分派任务 · sessions_send 工具
