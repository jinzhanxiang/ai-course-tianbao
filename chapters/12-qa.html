❓ 什么是 Loop Engineering？和 Harness 框架有什么区别？

🤖 决策者的回答

演进：Prompt（2023）→ Harness（2024-2025）→ Loop Engineering（2026-06 Anthropic 命名）
区别：Harness 管「单个 Agent」，Loop Engineering 管「Agent 的持续运行 + 记忆积累 + 自我进化」
5 元素：① 心跳 Automations（cron）② 知识沉淀 Skills ③ 记忆 Memory ④ 隔离 Worktrees ⑤ 判定 Eval
主公 OpenClaw：5 元素全部实现，是 Loop Engineering 的企业级完整实现。
来源：Boris Cherny（Anthropic）2026-06：「I personally wrote the Loop Engineering doc」

❓ 5 大 Agent 会不会抢活？怎么分工？

🤖 决策者的回答

不会抢，因为有 main 总调度。
• main 接指令 → 拆任务 → 分派给合适 Agent
• 每个 Agent 只做自己专业的事（data 清洗 / research 分析 / report 排版）
• 任何冲突由 main 仲裁
类比：main 是导演，5 Agent 是演员，演员不抢戏。
