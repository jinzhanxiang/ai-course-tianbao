📋 日报生成器

飞书群消息 → Agent 摘要 → 自动推送日报

提示词："读取最近 24 小时群消息，按主题分类摘要，不超过 500 字，Markdown 格式输出"

⏱ 搭建时间：10 分钟

🕷️ 信息爬取器

输入 URL → 自动爬取 → 结构化输出

提示词："爬取指定网页内容，提取关键数据和表格，输出结构化 JSON 并推送飞书"

⏱ 搭建时间：15 分钟

📊 数据清洗器

上传 Excel → 自动清洗去重 → 输出报告

提示词："读取上传的 Excel，去重、标准化日期格式、检测异常值，输出清洗报告"

⏱ 搭建时间：20 分钟

工作流本质 = 输入 → Agent（提示词 + 工具）→ 输出。你定义的是「输入什么、怎么处理、输出什么」。换项目名称，流程不变。

💬 Prompt 工程入门：如何与智能体高效对话

使用智能体的关键，在于学会「问对问题」——这是一项可以快速掌握的技能

📐 5 种核心提示词模式

1. 指令模式：「请用三段话总结以下内容...」

2. 角色模式：「你是一位有 10 年经验的新能源行业分析师...」

3. 示例模式：「参考以下格式输出：1.公司概况 2.财务分析 3.估值判断」

4. 步骤模式：「第一步：读取文件 第二步：提取关键数据 第三步：对比行业均值」

5. 约束模式：「不超过 500 字 · 使用表格对比 · 每个数据标注来源」

⚠️ 3 个常见 Prompt 错误

❌ 问题太模糊
→ 加角色 + 加约束，明确范围和深度
❌ 一次塞太多
→ 拆成多个子问题，逐个追问
❌ 不说输出格式
→ 给模板/示例/字数限制

📊 实战对比：差 Prompt vs 好 Prompt（星恒电源投资分析）

❌ 差 Prompt

「帮我分析一下星恒电源」

✗ 无角色定位
✗ 无分析维度
✗ 无输出格式
✗ 无数据来源要求

✅ 好 Prompt（角色 + 步骤 + 约束）

「你是一位新能源行业分析师。请基于星恒电源 FY24 审计报告（已上传），从以下三个维度分析其投资价值：
1. 行业地位（与宁德时代/比亚迪/中创新航对比）
2. 财务健康度（营收增速、毛利率、负债率）
3. 估值合理性（PE 对比、DCF 关键假设）
每个维度不超过 200 字，数据必须标注来源页数。」

📋 可复制的 Prompt 模板（投资行业专用）

🔬 行业研究

「你是一位[XX行业]分析师。请分析[行业名称]的竞争格局：
1. 市场规模与增速
2. 前5名企业市场份额
3. 技术替代风险
数据来源标注具体出处。」

💰 财务分析

「请基于[公司名]最新财报，从盈利能力、偿债能力、营运能力、成长能力四维度分析财务状况。用表格展示关键指标及同比变化，异常指标单独标注。」

📝 会议纪要

「请将以下会议录音/文字转为结构化纪要：
1. 会议主题
2. 参会方
3. 核心结论（3-5条）
4. 分歧点
5. 待办事项（含责任人和截止日期）
总字数不超过 800 字。」

🔧 常见使用障碍与破解

三个新人最容易卡住的地方，以及最简单的破解方法

障碍 1 数据从哪来？
公开数据
研报客API · Tavily搜索 · 天眼查
本地文件
PDF/Excel 拖拽上传 → OCR 提取
数据库
iFind · AKShare（A股/港股）
Agent 不会"自己找数据"——你需要告诉它去哪找
障碍 2 自动化怎么起步？
Level 1 入门
日报/周报自动生成（一段 Prompt）
Level 2 进阶
URL 爬取 → 结构化 → 推送飞书
Level 3 深化
Excel → 去重/标准化/异常检测 → 报告
原则：一次只自动化一个环节，跑通再加下一个
障碍 3 模型怎么选？
简单任务
摘要/翻译/格式转换
DeepSeek / GLM（免费）
主力任务
研究/分析/报告
MiniMax M3 / M2.7
复杂任务
估值建模 / Monte Carlo
Claude Sonnet / DeepSeek V4 Pro
敏感数据
oMLX 本地部署（数据不出机器）
三个障碍的本质： 数据源 = 知道"去哪找"而非等模型自己找 · 自动化 = 渐进式，从一段 Prompt 开始 · 模型选择 = 按任务复杂度分级匹配
