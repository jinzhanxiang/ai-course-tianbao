⚡ 破除迷思：你不需要训练模型

大多数人以为需要训练模型，实际上编排已有能力才是 90% 的真实场景

维度🎓 训练（模型微调）🛠️ 编排（提示词 + 工具）
投入$100K-1M 数据 + GPU免费开始
周期周/月1 小时
技能要求ML 工程师会用电脑即可
适用场景专业领域术语、安全合规日报/爬取/分析/摘要
维护成本持续数据标注改提示词即可
结论：编排是性价比最高的起点。只有当你有了明确的专业术语库、大量标注数据、且提示词优化已经到天花板时，才考虑训练。

🕷️ 信息爬取器

输入 URL → 自动爬取 → 结构化输出

提示词："爬取指定网页内容，提取关键数据和表格，输出结构化 JSON 并推送飞书"

⏱ 搭建时间：15 分钟

📊 数据清洗器

上传 Excel → 自动清洗去重 → 输出报告

提示词："读取上传的 Excel，去重、标准化日期格式、检测异常值，输出清洗报告"

⏱ 搭建时间：20 分钟

工作流本质 = 输入 → Agent（提示词 + 工具）→ 输出。你定义的是「输入什么、怎么处理、输出什么」。换项目名称，流程不变。
