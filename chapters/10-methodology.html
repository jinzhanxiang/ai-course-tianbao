核心理念

"纵深是把刀横着切，横向是把布竖着撕。只有纵横交叉，才能看清全貌。"

↕ 纵深：一个变量，算到极致

纵深解决的是："这个结论有多可靠？"——通过穷尽一个变量的所有可能性，找到置信区间。

🔢 典型纵深问题

  • WACC 敏感性：±2%，估值区间多大？
  • 退出倍数：5x/6x/7x，IRR 怎么变？
  • 永续增长率：2%/3%/4%，终值偏移多少？
  • 收入增速：±10%，NPV 敏感性？
  • Monte Carlo：10000 次模拟，概率分布？

💻 Claude Code 怎么做

Python 代码片段
import numpy as np
wacc_range = np.linspace(5.5, 9.5, 100)
dcf_values = [calc_dcf(w=w) for w in wacc_range]
plt.plot(wacc_range, dcf_values)
plt.axhline(y=45, color='r', linestyle='--')

Claude Code 自动写 Python → 运行 → 画敏感性曲线 → 输出置信区间。

人工 vs Claude Code： 手工算 WACC 敏感性需要 1 天；Claude Code 10 分钟出图+数字，效率提升 50 倍。

📐 纵横分析法总结

↕

纵深

工具：Claude Code
方法：敏感性分析、Monte Carlo、情景模拟
回答："结论有多可靠？"
产出：置信区间、概率分布图
↔

横向

工具：research V3
方法：7 维度穷举法、行业逻辑树
回答："还有哪些没想到？"
产出：维度清单、逻辑树图
交叉验证 = 纵深 × 横向

只做横向：全但不准。只做纵深：准但不全。
纵横交叉 = 又全又准。
