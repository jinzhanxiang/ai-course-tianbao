📌 Wiki 启动脚本内部（~/.hermes/scripts/start-wiki.sh）：

  • 启动 3 个服务：静态文件服务器 (19000) + API 服务器 (19001) + Cloudflare Tunnel
  • 自动 kill 旧进程 + 验证端口存活
  • 日志：/tmp/wiki-logs/{api,static,tunnel}.log

🔁 看门狗机制：每 5 分钟检查 3 个服务是否存活 → 挂掉自动重启 → 日志 /tmp/wiki-logs/watchdog.log
