协作流程
main 检查 → 冻结升级 → Claude Code 修复

🔧 故事 4：OpenClaw 帮 Hermes 修改配置

2026-05-08 · provider 大小写问题导致 Hermes 不认模型

问题现象

Hermes 配置文件中使用 provider: openai（小写），
但 Hermes 只认首字母大写的 provider: OpenAI

❌ 错误配置

provider: openai
model: deepseek-vash-flash

✅ 正确配置

provider: OpenAI
model: deepseek-vash-flash
结论： OpenClaw 帮 Hermes 排错，main 代理通过配置审查发现问题，修正后 Hermes 恢复正常。

  3. 📡 分派任务 · sessions_send 工具
  4. 📊 跟踪进度 · 心跳每 15min 巡检 · task-tracker.json
  5. 📋 验收反馈 · 闭环主公 · 失败立即告警
