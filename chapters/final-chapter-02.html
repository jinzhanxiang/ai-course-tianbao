🤖 什么是智能体？

先看清楚这场变革 —— 智能体时代 vs 传统时代，到底变了什么？

变革大背景：我们正在经历 「软件 2.0」 的变革。从「手工生产」→「智能工厂」需要三步：云计算 (2010s) → 大模型 (2023) → 智能体 (2024+)。

📊 智能体时代 vs 传统时代（5 个核心差异）

维度 传统软件时代（2000-2023） 智能体 Agent 时代（2024+） 变革
交互方式 点按钮 / 填表单 / 下拉菜单 说话下达目标 从 UI 到 NL
使用主体 人被软件指挥 人指挥软件干活 主体反转
输出类型 信息展示 / 表格汇总 执行结果 / 交付物 从查到做
调用关系 单个软件独立运行 多个 Agent 并行协同 + 反向调用人 从点到网
能力交付 「点击什么出什么」 「赋予一个可用的智能能力」 从能用到能动

🌍 拿 3 件事看变革

📧 发一封邮件

传统：打开 Outlook → 点「新建」→ 输入收件人 → 写主题 → 写正文 → 点「发送」
智能体：说话：「帮我给王总发一封邮件，说明天下午 3 点跟他确认星恒电源尽访谈」

📑 出一份尽调报告

传统：50 页 Word → 手工查财报 → 手工录数据 → 手工改格式
智能体：一句话：「出星恒电源尽调报告」→ 7 维度并行 90 分钟出 80 页成稿

🧾 清洗 13 份研报

传统：实习生请假 3 天手工 OCR + Excel
智能体：data agent 2 小时 → 79 实体 / 110 指标 / 178 逻辑 / 自动入库

⑤ 具身智能
💬
① ChatBot 时代
关键词：知识百科 · 单轮对话
一句话能聊，一件事做不了。不能读文件、不能跑代码、不能改数据库。
代表：ChatGPT、文心一言、通义千问
🪄
② Copilot 时代
关键词：贴身副驾 · 半自动
人在回路：AI 起草，人审核。提效 30-50%，但每一步都要人点头。
代表：GitHub Copilot、Cursor、Microsoft 365 Copilot
🤖
③ Agent 时代
关键词：目标驱动 · 全自动单兵
你定目标，它自己拆任务、调工具、交付结果。Agent = LLM + 工具 + 记忆 + 规划。
代表：Devin、AutoGPT、Manus、Computer Use
👥
④ 协同时代 ⭐
关键词：数字团队 · 联合作战
main 调度 + 多个专业 Agent（研究 / 数据 / 报告 / 项目）分工协同，完成单一 Agent 干不了的任务。
代表：OpenClaw · 本节课要讲的体系
🦾
⑤ 具身智能
关键词：物理世界 · 走进现实
Agent 从屏幕走进物理世界：自动驾驶、人形机器人、四足机器狗。通用人工智能 (AGI) 的关键拼图。
代表：Tesla Optimus、Figure 02、宇树 H1、小鹏 PX2
「从聊天到干活 · 从单兵到团队 · 从屏幕到物理世界」
我们正处在 第 ④ 阶段 的起点 —— 多 Agent 协同 + 长期记忆 + 垂直行业落地

🏗️ 智能体的最小架构

一个智能体 = 大脑 + 双手 + 记忆 + 计划 —— 缺一不可

🧠

大脑（LLM）

负责推理、决策、规划。
MiniMax / Kimi / Claude / Deepseek / 小米 Mimo，多模型融合。

🔧

双手（Tools）

读 PDF、写文件、跑 Python、调数据库。
Function Call 协议。

📝

记忆（Memory）

短期上下文 + 长期向量库。
OpenClaw 用 Qdrant + bge-m3。

📋

计划（Planning）

CoT 思维链 + ToT 思维树 + ReAct 推理-行动循环。
