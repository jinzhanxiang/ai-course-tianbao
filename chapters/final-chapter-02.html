Step 4 · 市占率 72.5%，为什么不赚钱？

这是整个尽调中最关键的问题——也是最终「不建议收购」结论的核心逻辑链

四层护城河侵蚀追踪（FY2020-2025）

1. OEM 关系瓦解
主机厂直接供货占比 80% → 6.38%（FY2020→FY2025）。雅迪+爱玛累计出货量极高但收入贡献仅 6.38%——ATL 新能安为主机厂提供了第二供方选择，星恒从「必选项」变成「可选项」
2. 产品结构低端化
核心收入仍来自低毛利 LMO 圆柱电芯，高毛利 LMFP 产能利用率仅 15-25%，远未达到盈亏平衡点（约 40-50%）
3. 前驱体体外循环
斯科兰德关联交易 + 体外循环结构——利润在体系内流转而非沉淀在星恒主体，详见附件E
4. 合规市场萎缩
两轮车锂电合规市场总量 874万→474万组（-45.8%）（FY2022→FY2025）。星恒 72.5% 的高市占率有分母效应——全口径市占率仅 16.7%

核心结论
星恒的高市占率掩盖了三个结构性问题：(1) 合规市场在快速缩小——分母变小，市占率被动上升；(2) 最赚钱的 OEM 渠道在瓦解——主机厂给了 ATL 第二供方地位；(3) 新增产能（青源 LMFP）的盈利拐点至少 2-3 年——短期无法改善利润。高市占率 ≠ 高盈利质量。

45 秒发现券商估值缺陷

国信证券 PE 均值 32.86x 含宁德时代——CATL 营收 4,237 亿 = 星恒的 116 倍，结构不可比。

智能编程助手在读取可比公司列表时，自动识别出 CATL 作为异常值——不是我们让它查的，是它自己发现的。

Agent 执行过程：

1. 读取附件H 可比公司列表（8 家公司 + PE 倍数）

2. 自动计算每家公司的营收规模

3. 发现 CATL 营收 4,237 亿 vs 其他 7 家均值 36.3 亿

4. 标记为「结构不可比」并建议剔除后重算

→ 剔除 CATL 后 PE 均值从 32.86x 降至 9.42x，差异 3.5 倍

高市占率悖论自动解构

人类直觉：「72.5% 市占率 = 强」。但 Agent 30 分钟内从四个维度系统解构了这个直觉。

不是我们让它「质疑市占率」——是在做横向对标时，它自己跑出来的发现：高市占率≠高盈利。

Agent 分析路径：

1. 提取 FY2020-2024 市占率 vs 毛利率 vs 净利率时间序列

2. 发现：市占率 45%→72.5%，毛利率同期 20%→8%——反向运动

3. 交叉验证：对比天能/超威——市占率更低但毛利率更高

4. 自动生成四层侵蚀框架（价格战/产品结构/客户集中/技术替代）

→ 核心洞察：星恒的市占率是靠降价抢来的，不是靠产品力赢来的
