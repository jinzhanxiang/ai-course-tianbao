⚡ 破除迷思：训练 vs 编排

大多数人以为需要训练模型，实际上编排已有能力才是 90% 的真实场景

维度 🎓 训练（Fine-tuning） 🛠️ 编排（Prompt + Tools）
投入$100K-1M 数据 + GPU免费开始
周期周/月1 小时
技能要求ML 工程师会用电脑即可
适用场景专业领域术语、安全合规日报/爬取/分析/摘要
维护成本持续数据标注改 Prompt 即可

结论：编排是性价比最高的起点。只有当你有了明确的专业术语库、大量标注数据、且 Prompt 优化已经到天花板时，才考虑训练。

2️⃣ Step 2：安装 Hermes — Windows 一键包

演讲结束后，拷贝 hermes-installer.zip，回家解压即用

📦 安装包包含

🖥️Hermes 主程序（Windows exe）
📝配置文件模板（config.yaml）
🔧setup.bat — 双击自动配置
🪶飞书插件（lark-messenger）
📋3 个预置工作流模板
📖README.html — 图文教程

⚡ 安装步骤

  1. 解压 hermes-installer.zip
  2. 双击 setup.bat
  3. 输入商汤 API Key
  4. 输入飞书 Webhook 地址
  5. 完成！hermes start 启动

无需 Python 环境、无需命令行基础。setup.bat 自动检测环境、自动安装依赖。

3️⃣ Step 3：连接飞书 — @机器人 即可调用

核心思路：在飞书群里创建一个自定义机器人 → Hermes 监听消息 → 自动回复。所有同事都能用。

🔗 配置步骤

  1. 飞书群 → 设置 → 群机器人
  2. 添加自定义机器人 → 复制 Webhook URL
  3. 在 Hermes 配置中填入 Webhook
  4. hermes plugin enable lark

💬 使用方式

群里 @机器人：
"帮我总结今天的重要消息"

4️⃣ Step 4：定义你的第一个工作流

📋 日报生成器

飞书群消息 → 智能体摘要 → 自动推送日报

Prompt:
"读取最近 24 小时群消息，按主题分类摘要，不超过 500 字，Markdown 格式输出"

⏱ 搭建时间：10 分钟

🕷️ 信息爬取器

输入 URL → 自动爬取 → 结构化输出 → 飞书推送

Prompt:
"爬取指定网页内容，提取关键数据和表格，输出结构化 JSON 并推送飞书"

⏱ 搭建时间：15 分钟

📊 数据清洗器

上传 Excel → 自动清洗去重 → 输出分析报告

Prompt:
"读取上传的 Excel，去重、标准化日期格式、检测异常值，输出清洗报告"

⏱ 搭建时间：20 分钟

工作流本质 = Input → Agent（Prompt + Tools）→ Output。你定义的是"输入什么、怎么处理、输出什么"。
