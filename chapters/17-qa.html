❓ 5 大 Agent 会不会抢活？怎么分工？

🤖 决策者的回答

不会抢，因为有 main 总调度。
• main 接指令 → 拆任务 → 分派给合适 Agent
• 每个 Agent 只做自己专业的事（data 清洗 / research 分析 / report 排版）
• 任何冲突由 main 仲裁
类比：main 是导演，5 Agent 是演员，演员不抢戏。
